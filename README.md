Сборка учебных проектов на Python:

"Вероятностные методы криптографии"
Реализован полный цикл анализа булевых функций

- Генерация случайной подстановки `g: V₆ → V₆`
- Вычисление **веса Хэмминга**
- Построение **полинома Жегалкина** (алгебраическая нормальная форма)
- Анализ **сбалансированности** функции
- Поиск **запрещённых последовательностей**
- **Спектральный анализ** (преобразование Уолша-Адамара)
- Поиск **наилучших линейных аппроксимаций**
- Вычисление **автокорреляции**
- Оценка **k-стабильности**
